Features of HTML5

Semantic Elements

HTML5 introduces new semantic elements such as <header>, <footer>, <article>, <section>, and <nav> which provide a clearer structure to web pages, making them more accessible and SEO-friendly.

Canvas and SVG

The <canvas> element enables drawing graphics and animations directly within the browser using JavaScript. SVG (Scalable Vector Graphics) allows for more scalable and resolution-independent graphics.

Audio and Video Support

HTML5 includes native support for embedding audio and video content into web pages using the <audio> and <video> elements. This eliminates the need for third-party plugins like Flash.
